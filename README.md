## Setup

### Requirements

* Python 3.11+

### Installation

Install it directly into an activated virtual environment:

```text
$ pip install metr-cli
```

or add it to your [Poetry](https://poetry.eustace.io/) project:

```text
$ poetry add metr-cli
```

## Usage

After installation, the package can be imported:

```text
$ python
>>> import metr_cli
>>> metr_cli.__version__
```
